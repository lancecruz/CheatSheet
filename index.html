HTML Table Reference

Table Tags

Tag Name Description
<table> Table Defines a table
<thead> Table Header Groups the header content in a table
<tbody> Table Body Groups the body content in a table
<tfoot> Table Footer Groups the footer content in a table
<tr> Table Row Defines a row in a table
<th> Table Heading Defines a header cell in a table
<td> Table Data Defines a cell in a table
<caption> Table Caption Defines a table caption

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a <td> element should span.
rowspan Row Span Defines how many rows a <td> element should span.

CSS Selectors

Format Name Description
element Type Selector Matches every instance of the element.
#id ID Selector Styles the element with the specified id.
.class-name Class Selector Selects HTML elements with a specific class attribute.
* Universal Selector Selects all HTML elements on the page.
h1, h2, p Grouping Selector Selects all the HTML elements with the same style definitions.
element.class-name element.class Selector Select and style every element with the specified class.
element element Descendant Selector Made up of two or more selectors separated by white space.
element > element Child Selector Select and style every element when an element is the child of some element.
element1 + element2 Adjacent Siblings Selector The selector matches if element1 and element2 share the same parent in the document tree and element1 immediately precedes element2.
element[attribute], element[attribute=val] Attribute Selector Matches elements which have certain attributes.
